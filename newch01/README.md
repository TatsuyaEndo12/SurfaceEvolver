## ベシクルモデルの作成（曲げ弾性エネルギーを定義）
ベシクルの平衡形状は曲げ弾性エネルギーを最小化する形状として計算できる。
プログラム上では後述の{V;u;g}というコマンドでこの計算が実行される。
